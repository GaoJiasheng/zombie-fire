Zombie Fire 隐私政策

最后更新：2026 年 7 月 17 日

摘要：Zombie Fire 当前版本不会收集、上传、出售或共享个人数据，也不包含广告、分析、账号、内购或第三方追踪。

1. 数据收集

Zombie Fire 是一款可离线游玩的单机游戏。当前版本不收集个人信息、设备标识符、位置、通讯录、照片、使用情况或诊断数据，也不会将游戏数据传输到开发者服务器。

2. 本地存档

关卡星级、金币、经验、角色和装备选择、武器升级及设置等游戏进度只保存在玩家设备本地。该数据不会自动上传或与第三方共享。

3. 删除数据

玩家可以在游戏设置中使用“重置存档”删除游戏进度，也可以通过卸载应用移除设备上的全部本地游戏数据。由于开发者不持有玩家数据，因此没有服务器端账户或数据需要另行删除。

4. 广告、分析与第三方服务

当前版本不包含广告 SDK、分析 SDK、账号登录、应用内购买、推送通知、云存档或第三方追踪服务。

5. 儿童隐私

本游戏不会主动向儿童或其他玩家收集个人数据。

6. 政策变更

如果未来版本新增联网服务、分析、广告、购买、排行榜、云存档或账号功能，本政策和 App Store 隐私披露将在相关版本发布前更新。

7. 联系方式

如对本政策有疑问，请发送邮件至 gaojiasheng.him@foxmail.com，或通过 Zombie Fire 支持页面联系我们。

Zombie Fire Privacy Policy

Last updated: July 17, 2026

Summary: The current version of Zombie Fire does not collect, upload, sell, or share personal data. It contains no advertising, analytics, accounts, in-app purchases, or third-party tracking.

1. Data Collection

Zombie Fire is a single-player game that can be played offline. The current version does not collect personal information, device identifiers, location, contacts, photos, usage data, or diagnostics, and it does not transmit gameplay data to a developer-operated server.

2. Local Save Data

Game progress—including level stars, gold, experience, character and equipment selections, weapon upgrades, and settings—is stored only on the player's device. This data is not automatically uploaded or shared with third parties.

3. Deleting Data

Players can delete progress with the Reset Save option in Settings or remove all local game data by uninstalling the app. Because the developer does not hold player data, there is no server-side account or data record that requires a separate deletion request.

4. Advertising, Analytics, and Third Parties

The current version includes no advertising SDKs, analytics SDKs, account login, in-app purchases, push notifications, cloud saves, or third-party tracking services.

5. Children's Privacy

The game does not knowingly collect personal data from children or any other players.

6. Changes to This Policy

If a future version adds online services, analytics, advertising, purchases, leaderboards, cloud saves, or accounts, this policy and the App Store privacy disclosure will be updated before that version is released.

7. Contact

For questions about this policy, email gaojiasheng.him@foxmail.com or use the Zombie Fire support page.
