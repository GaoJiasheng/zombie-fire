# 07 · 元素克制 · 20 僵尸 · 8 Boss

## 1. 元素克制系统

5 元素：`physical 物理` / `fire 火` / `ice 冰` / `lightning 雷` / `poison 毒`。

不做复杂的环形相克，而是**每个敌人挂"弱点/抗性/免疫"标签**（简单、可读，符合"看得懂"支柱）。核心原则：**弱点是推荐解，抗性是阻力，免疫是 Boss 机制，不是普通关卡门票**。

| 关系 | 倍率 | 含义 |
|---|---|---|
| 弱点 weakness | ×1.5 | 该元素打它更疼 |
| 普通 normal | ×1.0 | 无修正 |
| 抗性 resist | ×0.5 | 该元素打它半伤 |
| 免疫 immune | ×0 | 完全无效（主要用于 Boss） |

每个普通敌人可有 1 个弱点 + 0~1 个抗性，普通敌人不做硬免疫；Boss 可有 1 个免疫或阶段免疫，但必须在选关、Boss 登场和命中反馈中三重提示。
关卡选择界面**明示当关主要威胁与推荐弱点**（参考《向僵尸开炮》"关卡标注弱点"），让玩家赛前配元素；但非推荐元素仍可通关，只是更难满星。

元素附带效果回顾（详见 `05`）：火=灼烧DoT、冰=减速/冻结、雷=连锁/麻痹、毒=叠层DoT(可按%最大生命)、物理=无附带但稳定通用，少数特殊 Boss 会要求物理输出窗口。

### 威胁标签 `threat_tag`

敌人额外标注目标优先级标签，驱动自动瞄准、追踪弹、宝宝 AI 和 HUD 标记：

| 标签 | 说明 | 示例 |
|---|---|---|
| `breach` | 高近线威胁，到达防线后会快速攻击基地 | runner / charger / berserker |
| `support` | 给敌方加速、护盾、复活 | screamer / necromancer / warden |
| `burst` | 自爆或短时大伤害 | bomber |
| `tank` | 高血/吸火力 | brute / juggernaut |
| `window` | 有输出窗口或虚化窗口 | phantom / void_phantom |

- 8 个 Boss 系统性覆盖"5 种免疫 + 全免疫 + 再生 + 轮换"，构成完整的"build 检验"，但每个 Boss 在首次出现前必须通过普通关/支线提示给出准备路径。
- `boss_void_phantom`（全免疫→物理）是 `vanguard` 物理路线的高光；`boss_apex_overlord`（轮换）是对玩家整体养成与应变的终极考核。两者都是明确告知的考试，不做突然翻桌。
- Boss 关均含前置普通波次铺垫，再进入 boss wave（见 `08`）。

## 4. 精英怪机制

普通波次中会出现**精英版**（词缀强化的普通僵尸）：
- 视觉：描边发光（见 `11`），体型略大。
- 数值：HP/伤害提升，掉落更多金币与 run_xp。
- **击杀精英额外触发一次局内三选一**（见 `03`），是节奏与战术高点。
- 词缀示例：迅捷（更快）、坚韧（更肉）、剧毒（死亡毒云）、护盾（自带护盾）——复用机制资产，低成本扩展难度。
