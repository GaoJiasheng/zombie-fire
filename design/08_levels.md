# 08 · 关卡设计（99 关）

> 10 段关卡背景，每 10 关切换一张主战斗背景；Boss 和怪物梯队仍按 4 个大章节推进。每关有建议等级、波次脚本、三星标准。
> 全部关卡参数走 `levels.json`（结构见 `data/schema.md`）；本文给设计框架、难度曲线与代表性关卡示例。

## 2. 建议等级曲线

每关 `recommend_level`：第 1 关建议 Lv1，第 99 关建议 Lv≈80。整体近似线性微指数：

```
recommend_level(n) ≈ round( 0.8 * n )   // 粗略：第n关建议等级≈0.8n
boss 关 +2~3 级建议（机制更硬）
```

| 关 | 建议等级 | 关 | 建议等级 |
|---|---|---|---|
| 1 | 1 | 60(Boss) | 50 |
| 12(Boss) | 12 | 72(Boss) | 60 |
| 24(Boss) | 22 | 84(Boss) | 70 |
| 36(Boss) | 32 | 99(Boss) | 80 |
| 48(Boss) | 42 | | |

- 低于建议等级：可进，标黄"有挑战"，更吃操作（可能只 1 星）。
- 达到建议等级 + 核心装备按 `09` 强化档：合理操作可 2~3 星。

## 3. 难度三维度（不靠单纯堆数值）

每关难度 = **数值缩放 × 机制组合 × 节奏强度** 三者综合：

1. **数值缩放**：怪物 HP/伤害随关卡用统一公式放大（见 `09`），与建议等级对齐。
2. **机制组合**：登场怪种类与混编复杂度递增（前期单一→后期多机制同屏）。
3. **节奏强度**：波次密度、同屏怪量、入场速度。Boss 关与"狂潮关"节奏陡升。

### 特色关型（穿插，避免单调）

| 关型 | 说明 | 出现 |
|---|---|---|
| 标准关 | 5~8 波常规推进 | 多数 |
| 狂潮关 | 短时间海量低血怪，考验清群（split/tesla 流派爽点） | 每章 1~2 个 |
| 精英关 | 多个精英怪混编，节奏高、奖励高 | 每章 1~2 个 |
| 限定弱点关 | 全场怪共享同一弱点，鼓励专精元素（也提示"换甲/换炮"） | 每章若干 |
| Boss 关 | 前置普通波 + boss wave | 每章 2 个 |
| 爽感关 | 给低血密集尸潮和高权重弹幕牌，专门让玩家体验清屏 | 前 20 关密集出现，后续穿插 |
| 变异挑战关 | 同一关换特殊词缀，主线外可重复挑战 | M3/M4 后开放 |

弱点关必须按"推荐解"设计：带对元素更容易三星，带错元素仍可首通。硬免疫只出现在 Boss 关，且在选关界面、加载提示、命中反馈里都明示。

## 4. 波次脚本结构（levels.json）

```jsonc
{
  "id": "level_024",
  "env": "env_abandoned_factory",
  "chapter": 1,
  "recommend_level": 22,
  "difficulty_coef": 2.4,        // 进入 09 的缩放公式
  "primary_weakness": "ice",     // 选关界面展示（本关主敌弱点）
  "base_hp_ref": 100,            // 该关基地血量基准
  "waves": [
    { "wave": 1, "spawns": [
        {"type":"zombie_runner","count":8,"interval":0.6,"lane":"spread"},
        {"type":"zombie_bomber","count":2,"interval":1.5,"lane":"center"} ] },
    { "wave": 2, "spawns":[ ... ] },
    { "wave": "boss", "boss":"boss_inferno_maw", "support":[
        {"type":"zombie_shambler","count":6,"interval":2.0} ] }
  ],
  "star_rule": "base_hp_percent",   // 三星=满血,二星>=50%,一星>0
  "first_clear_reward": {"gold": 500},
  "first_3star_reward": {"star_bonus": 0, "drop": "armor_thermal_shard"}
}
```

> 字段定义见 `data/schema.md`。设计师改关卡只改 JSON，不动代码。

## 5. 三星标准（统一规则，见 `01`/`02`）

按**通关时基地剩余血量百分比**：100% = ★★★，50–99% = ★★，>0–49% = ★。
回刷可补星（只补差额）。Boss 关同规则，但因机制更硬，满星更考验"对路 build + 操作"。

## 6. 代表性关卡示例

### Lv1 教学关（city_ruins）
- 建议 Lv1，只出 `zombie_shambler` 慢怪，3 波。
- 引导：拖动瞄准、自动开火、第一次三选一、看懂"弱点=火"。
- 必拿 3 星（教学不该劝退）。

### Lv2~5 爽感引导
- Lv2：给 `skill_split_shot` 高权重，尸潮低血成群，让玩家第一次看到弹幕分裂清屏。
- Lv3：引入 `zombie_runner`，教学目标策略"越线威胁优先"。
- Lv4：给 `skill_pierce` / `skill_multishot` 高权重，教学穿透/多重射击的质变。
- Lv5：第一只精英怪，击杀后额外三选一，建立"精英=节奏高点"。

### Lv6~10 系统分层开放
- Lv6：开放护甲/芯片基础强化。
- Lv7：引入推荐弱点，但不要求换装。
- Lv8：引入主动技能 CD 与技能槽高亮。
- Lv9：开放宝宝，展示自动补刀和金币吸入。
- Lv10：第一场小型狂潮关，允许玩家用前面学到的弹幕组合爽一次。

### Lv12 首领·钢铁泰坦（免疫物理）
- 前置 3 波 T1 普通怪铺垫。
- boss wave：tank_titan 周期召装甲僵尸。
- **教学点**：物理打不动它（免疫物理），引导玩家第一次理解"换元素/带元素技能"。选关界面提示弱点=雷，并在前 11 关给过雷/元素附加的获取路径。
- 建议 Lv12 + 核心装备对应档可 2~3 星。

### Lv36 首领·冰封领主（周期冻你的炮塔）
- **教学点**：炮塔会被冻住数秒攻速归零 → 价值凸显**宝宝/DoT/护盾**（被冻期间靠宝宝和灼烧续命）。提示弱点=火。

### Lv72 首领·虚空幽帝（全元素免疫）
- **教学点**：所有元素无效，**需要物理输出窗口**。引导玩家提前准备 `vanguard` 物理炮或物理装。相位虚化期需停火等输出窗口（操作考验）。

### Lv99 终局·终焉霸主（轮换免疫，多阶段）
- 多阶段轮换免疫元素 + 召唤历代 Boss 缩小机制。
- 满养成 + 灵活换装/换角色 + 操作三者皆需，作为整个游戏的"毕业考"。
- 设计为"达到建议等级 Lv80 且装备到位的玩家，多尝试几次可通"，3 星为高玩追求。

## 7. 关卡产出对齐（防卡墙）

- 每关金币产出（怪掉 + 首通）对齐"在本关或前关重刷 2~4 遍 → 攒够把核心装备升到下一档"（见 `09`）。
- 星产出（最多 3/关）对齐解锁节奏：打到某章自然攒够星解锁该章需要的新角色/新元素主炮。
- **保证**：任何关卡都不存在"非氪/非肝到天荒地老不可过"的硬墙；难点永远是"机制+操作+对路 build"，而非纯数值。

## 8. 前 20 关开放节奏

| 关段 | 开放/教学 | 设计目标 |
|---|---|---|
| 1–5 | 瞄准、自动开火、三选一、分裂/穿透、精英奖励 | 先爽，少解释 |
| 6–10 | 装备强化、弱点提示、主动技能、宝宝 | 建立"每局后变强" |
| 11–12 | Boss 准备、元素替代解、首个硬免疫 | 让玩家理解配装考试 |
| 13–16 | 芯片组合、目标策略切换、威胁标签 | 解决"该打谁" |
| 17–20 | 第一章中型狂潮/精英混编 | 让玩家主动回刷满星 |

M1 只做 1–5 的缩小版，但必须验证这张节奏表：玩家在 3 分钟内看到清屏、精英、选卡质变和结算奖励。
