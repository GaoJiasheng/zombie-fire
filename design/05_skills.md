# 05 · 技能设计（16 通用 + 8 专属）

> 携带规则、两层升级、三选一逻辑见 `03`。本文定义每个技能的效果与 **Lv1~Lv5 成长分支树**。
> 数据结构见 `data/schema.md` 的 `skills.json`。每个技能成长树骨架统一为：Lv1 基础 → Lv2 量变 → Lv3 分支A/B → Lv4 强化 → Lv5 终极X/Y（详见 `03` §3）。

## 携带速记

- 5 槽 = 2 专属（固定） + 3 通用（自带1 + 局内加权成型2）。
- 通用技能需 `base_level≥1` 才可被携带 / 被局内选中。
- 技能分**主动**（点击触发，有 CD）与**被动**（携带即生效）。
- 每个技能有 `tags`，卡牌导演用它做加权：`projectile / element / control / defense / economy / tempo / boss / anti_swarm / anti_armor / execute`。
- 技能设计必须保留"质变瞬间"：Lv3 分支和 Lv5 终极尽量改变打法或视觉反馈，而不只是 +10% 数值。

## A. 通用技能池（16）

### A1 弹道/输出类（8）

#### `skill_split_shot` 分裂弹（被动）
子弹命中后分裂成多枚小子弹向前扩散。
- Lv1 命中分裂 2 枚（伤害衰减）。 Lv2 分裂 3 枚 / 衰减降低。
- **Lv3 A 散射**：分裂数再+，扇面更广（清群）。 **B 聚焦**：分裂数少但每枚伤害高（点杀）。
- Lv4 延续所选方向数值强化。
- **Lv5 X 链式分裂**（分裂子弹可再分裂一次）/ **Y 穿透分裂**（分裂子弹带 1 穿透）。

#### `skill_pierce` 穿透弹（被动）
子弹可穿透多个敌人。
- Lv1 穿透 1。 Lv2 穿透 2。
- **Lv3 A 不衰减**（穿透不掉伤）/ **B 撕裂**（每穿一个，目标受伤+）。
- Lv4 穿透 +1~2。
- **Lv5 X 贯穿**（穿透无上限/全程不衰减）/ **Y 钉刺**（被穿透敌人短暂减速）。

#### `skill_ricochet` 弹射（被动）
子弹命中后弹向附近敌人。
- Lv1 弹射 1 次。 Lv2 弹射 2 次。
- **Lv3 A 增幅弹射**（每次弹射伤害递增）/ **B 广域弹射**（弹射搜索范围大）。
- Lv4 弹射次数+。
- **Lv5 X 暴风弹射**（弹射次数大幅↑）/ **Y 回旋**（可弹回已命中目标补刀）。

#### `skill_multishot` 多重射击（被动）
每次开火额外发射子弹。
- Lv1 +1 弹（小角度散开）。 Lv2 +2 弹。
- **Lv3 A 平行**（多弹平行、密度高）/ **B 扇形**（多弹扇面、覆盖广）。
- Lv4 +1~2 弹。
- **Lv5 X 弹幕**（额外弹数最大化）/ **Y 集束**（多弹命中同点叠伤）。

#### `skill_homing` 追踪弹（被动）
子弹自动追踪敌人。
- Lv1 弱追踪。 Lv2 追踪转向更强。
- **Lv3 A 优先高威胁**（自动锁精英/法师/越线威胁，服从目标策略）/ **B 群追**（分配给不同目标）。
- Lv4 追踪精度/转速↑。
- **Lv5 X 锁定核心**（永远咬死最高威胁直至死亡）/ **Y 蜂群**（追踪子弹数翻倍）。

#### `skill_salvo` 齐射（主动，CD）
短时间内炮塔进入高速齐射。
- Lv1 攻速+150% 持续 3s。 Lv2 持续/增幅↑。
- **Lv3 A 风暴齐射**（齐射期额外散射）/ **B 精准齐射**（齐射期暴击率↑）。
- Lv4 CD↓ / 增幅↑。
- **Lv5 X 终焉齐射**（齐射期所有命中必暴）/ **Y 持久齐射**（持续时间大幅↑）。

#### `skill_critical` 致命一击（被动）
提升暴击率与爆伤。
- Lv1 +暴击率。 Lv2 +爆伤。
- **Lv3 A 狙杀**（对高血目标暴击额外加成）/ **B 连暴**（暴击后短时暴击率↑）。
- Lv4 暴击/爆伤继续↑。
- **Lv5 X 处决**（暴击对低血目标即死判定）/ **Y 会心风暴**（暴击溅射小范围暴击伤害）。

#### `skill_charge_shot` 蓄能炮（主动，CD）
蓄力发射一发高伤穿透重弹。
- Lv1 重弹 ×5 伤害、穿透 3。 Lv2 伤害/穿透↑。
- **Lv3 A 聚能**（命中爆发范围伤害）/ **B 贯日**（穿透全程、超远射程）。
- Lv4 蓄力更快 / 伤害↑。
- **Lv5 X 歼灭炮**（命中点超大范围爆发）/ **Y 轨道炮**（一线贯穿全屏）。

### A2 元素附加类（4）—— 给子弹附带元素，撑流派

#### `skill_incendiary` 燃烧弹（被动·火）
命中附加灼烧 DoT。
- Lv1 灼烧 3s。 Lv2 DoT 伤害/时长↑。
- **Lv3 A 烈焰**（DoT 可叠层）/ **B 焚风**（灼烧可向相邻敌人蔓延）。
- Lv4 DoT↑。
- **Lv5 X 地狱火**（灼烧致死引爆）/ **Y 永燃**（DoT 不再随时间衰减）。

#### `skill_cryo` 冰霜弹（被动·冰）
命中附加减速，叠满冻结。
- Lv1 命中减速。 Lv2 减速↑、叠层更快。
- **Lv3 A 速冻**（更快冻结）/ **B 霜甲碎**（被减速敌人受伤+）。
- Lv4 控制↑。
- **Lv5 X 绝对冻结**（命中概率瞬冻）/ **Y 冰爆**（冻结目标死亡冰爆减速一片）。

#### `skill_tesla` 感应电弧（被动·雷）
命中触发短链电弧。
- Lv1 弹跳 2 个。 Lv2 弹跳/伤害↑。
- **Lv3 A 过载弧**（弹跳更多）/ **B 导电**（命中目标受雷伤+）。
- Lv4 链数↑。
- **Lv5 X 雷网**（同时电多目标）/ **Y 麻痹**（电弧附带短麻痹）。

#### `skill_venom` 剧毒弹（被动·毒）
命中叠加中毒层，按层数持续掉血。
- Lv1 毒 5 层上限。 Lv2 层上限/每层伤害↑。
- **Lv3 A 剧毒**（层数越高 DoT 越狠）/ **B 腐蚀**（中毒敌人减防）。
- Lv4 上限/伤害↑。
- **Lv5 X 瘟疫**（中毒死亡向周围扩散毒云）/ **Y 噬体**（按目标最大生命%掉血，克高血/Boss）。

### A3 功能/防御/经济类（4）

#### `skill_barrier` 能量护盾（被动）
为基地提供护盾值，吸收越线伤害并缓慢回复。
- Lv1 护盾上限+ / 缓回。 Lv2 上限/回复↑。
- **Lv3 A 反射**（护盾抵挡时弹反小伤害）/ **B 急救**（破盾后短时大量回盾）。
- Lv4 上限↑。
- **Lv5 X 壁垒**（护盾极高、回复快）/ **Y 相位**（护盾存在时基地免疫一次致命越线）。

#### `skill_slow_field` 减速力场（主动/被动）
在基地防线前生成减速区。
- Lv1 防线前减速区。 Lv2 区域/减速↑。
- **Lv3 A 泥沼**（减速更强、可叠加）/ **B 荆棘**（区域内持续小伤害）。
- Lv4 范围↑。
- **Lv5 X 时间停滞**（区域内概率定身）/ **Y 绞杀场**（区域伤害大幅↑）。

#### `skill_gold_rush` 淘金热（被动·经济）
提升金币掉落与自动拾取。
- Lv1 金币掉落+ / 自动吸金。 Lv2 掉落%↑。
- **Lv3 A 暴富**（概率掉双倍金币）/ **B 磁化**（全屏自动吸金）。
- Lv4 掉落↑。
- **Lv5 X 点金**（暴击击杀掉额外金币）/ **Y 寻宝**（小概率掉装备强化材料）。
- 注：经济技能不增强战力，给"刷子流"玩家一个可选方向，不进入战力平衡。卡牌导演在危机局面会降低经济卡权重，避免玩家快漏怪时还被推金币牌。

#### `skill_recycle` 能量回收（被动·节奏）
击杀返还局内经验/缩短主动技 CD。
- Lv1 击杀少量返还 run_xp。 Lv2 返还↑。
- **Lv3 A 充能**（击杀缩短主动技 CD）/ **B 滚雪球**（连杀提升 run_xp 获取）。
- Lv4 效率↑。
- **Lv5 X 永动**（精英击杀直接给一次三选一）/ **Y 超频**（主动技 CD 大幅↓）。
- 注：这是"节奏放大器"，让会打的玩家更快成型，呼应 `03` 的节奏制升级。

## C. Synergy（流派搭配示例，给玩家"组合的快乐"）

| 流派 | 核心搭配 | 效果 |
|---|---|---|
| 分裂穿透流 | split_shot + pierce + multishot | 满屏弹幕，清群之王 |
| 灼烧瘟疫流 | blaze + incendiary + venom(瘟疫) | DoT 叠满，专克高血/再生/Boss |
| 冰封碎裂流 | frost + cryo + charge_shot | 冻住再一炮碎，控杀冲锋/快怪 |
| 雷霆连锁流 | volt + tesla + multishot | 越密越炸，秒装甲群 |
| 暴击处决流 | critical + salvo + railvolley | 高频暴击 + 处决斩杀 |
| 龟缩刷钱流 | barrier + slow_field + gold_rush | 稳过低风险关、屯金币 |

> 每个流派都有"克制的怪/Boss"与"被克的对象"，与 `07` 的弱点/免疫体系咬合，鼓励多 build。

## D. 卡牌导演权重标签

每张技能卡在 `skills.json` 中标注 `card_tags` 与 `weight_rules`，让三选一既可控又有惊喜：

| 标签 | 提权场景 | 示例 |
|---|---|---|
| `anti_swarm` | 狂潮关、低血怪多、同屏数量高 | split_shot / multishot / tesla |
| `anti_armor` | 装甲/持盾/高抗性怪多 | pierce / venom / charge_shot |
| `control` | 快怪、自爆、冲锋、漏怪压力高 | cryo / slow_field / barrier |
| `boss` | Boss wave 前后、单体高血目标 | charge_shot / critical / venom |
| `tempo` | 玩家连杀、主动技多、想快速成型 | recycle / salvo |
| `economy` | 低风险关、已领先、刷钱重玩 | gold_rush |

规则边界：
- 每次三选一至少 1 张顺 build 牌、至多 1 张纯经济牌。
- 连续 2 次没有出现同一核心 tag 时，下次提高保底权重。
- 低概率保留"离谱但有趣"的跨流派牌，制造 Roguelike 惊喜。
