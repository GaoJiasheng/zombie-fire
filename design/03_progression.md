# 03 · 养成系统与技能升级逻辑

> 本文是整个游戏长线乐趣的核心，重点定义你特别强调的 **技能两层升级逻辑**，并保证它与货币体系闭环、不会出现"升不动"或"无脑碾压"。

## 1. 养成总览（局外多条线）

| 线 | 资源 | 提升什么 |
|---|---|---|
| 角色等级 | 经验 `xp`（自动） | ATK / base_hp 基础盘；每级产出天赋点 |
| 技能成长 | 天赋点 `talent_point` | 技能**初始等级** + 解锁**成长分支** |
| 装备强化 | 金币 `gold` | 主炮/护甲/芯片/宝宝的数值 |
| 内容解锁 | 星 `star` | 新角色 / 新主炮 / 新技能进池 / 皮肤 |
| 挑战收集 | 星盈余/挑战奖励 | 皮肤、徽章、无限尸潮记录、图鉴 |

这些线分别吃不同资源、互不挤占，玩家每局结束都有"好几样在变强"的正反馈。

## 2. 技能两层升级逻辑（核心）

技能有 **两层等级**：局外"初始等级"（持久）+ 局内"临时等级"（关束清零）。

### 2.1 局外：技能初始等级（持久，吃天赋点）

- 每个技能（含角色专属技）有 `base_level`，范围 **Lv0~Lv5**。
- 用 `talent_point` 在【天赋/技能】界面升级。每升一级：
  - 提升该技能进场时的**起始强度**（出门即带这个等级的效果）。
  - 到关键级**解锁成长分支**（见 §3），让它在局内能进化的路线更多。
- 角色**专属天赋技**：出门**强制携带**、初始等级即 `base_level`，照常可升。
- 通用技能：`base_level ≥ 1` 才能被选入携带栏（Lv0 = 未学，需先花天赋点学第 1 级，或用星解锁进池）。

> **意义**：局外投资决定"你这套技能的下限和天花板"。投得多的技能，进场更强、局内能进化的花样更多。

### 2.2 局内：临时等级（关束清零，吃局内经验）

进入关卡后，技能可在本局内继续升（`run_level`），打完即重置回 `base_level`。

**携带与填槽规则（最终版）：**
- 每关共 **5 个技能槽**。
- **2 格 = 角色专属天赋技**：固定占用、出门即带。
- **3 格 = 通用技能**，采用「**带 1 + 局内成型 2**」：
  - 出门前玩家从已学通用技能里**自带 1 个**（放进第 1 个通用格）。
  - 另外 2 个通用格**进关时为空**，靠局内"三选一"填上。候选是加权随机，不是全随机，也不是完全点菜。

> 这样既有**确定的 build 意图**（2 专属 + 1 自带 = 你这局的核心流派已定），又保留 Roguelike 的**局内变数与惊喜**（剩下 2 格可能顺着流派，也可能歪出一套临时神 build）。

> **M1 可玩版落地说明**：角色专属不再挤占局内通用技能条。每个角色拥有 1 个独立主动按钮，1 个自动/弹种被动，并通过 `characters.json.active_skill` 与 `characters.json.bullet_affinity` 进入战斗；局内三选一卡仍负责通用技能成长。

### 2.3 局内升级的触发：按"节奏"而非"时间"

我们用**战斗节奏**驱动升级，不用现实时间（理由：按时间会惩罚稳扎稳打、且无视战斗表现；按节奏更公平、自然适配关卡长短）。

- 每只僵尸死亡给 **局内经验 `run_xp`**（精英/Boss 更多）。
- `run_xp` 攒满一管 → 触发一次 **「三选一」**。每管阈值随本局已升次数递增（前期快、后期慢）。
- **额外触发点**：每击杀一个**精英怪/小 Boss**，立刻额外给一次三选一（制造节奏高点与战术抉择）。

### 2.4 「三选一」给什么牌

系统按当前局面动态出 3 张牌，使用**卡牌导演 Card Director**，目标是"有倾向、有救场、有惊喜"：

1. **空槽未满时**：优先出"可加入的新通用技能"卡 → 选了就把这个技能放进一个空的通用格。
   - 候选池 = 玩家**已学（base_level≥1）但本局未携带**的通用技能。
   - 候选权重由 4 个因素决定：角色元素倾向、已带技能 tag、当前关主要弱点/威胁、上一次没选的保底权重。
   - 保证至少出现 1 张可填槽卡，直到 3 个通用格填满；另 1~2 张可为升级/通用增益，保留惊喜。
2. **槽已满后**：出"升级在场技能"卡 → 选中的技能 `run_level +1`，沿其**成长分支**走。
   - 到达分支节点时，卡面直接呈现"进化二选一"（如 分裂弹 Lv3：更多分裂 / 分裂附火）。
   - 可选到的分支由该技能**局外 base_level** 决定（局外没解锁的分支，局内不会出现）。
3. **偶发**：少量"通用增益卡"（如本局 +10% 攻、+1 暴击层）作为调味，权重低。

玩家可使用三种轻控制，避免被烂牌卡住，同时不抹平随机：
- **刷新 reroll**：每局基础 1 次；精英击杀/技能可额外给 `reroll_charge`。
- **锁定 pin**：把 1 张牌保留到下次三选一，适合等资源或等空槽。
- **跳过 skip**：获得少量金币/run_xp 补偿，不强迫拿废牌。

### 2.5 选卡权重示例

| 局面 | 高权重卡 | 低但保留的惊喜卡 |
|---|---|---|
| blaze + 火主炮 + 再生怪多 | incendiary / venom / slow_field | ricochet / critical |
| vanguard + 物理炮 + 装甲怪多 | pierce / tesla / charge_shot | gold_rush |
| 快怪即将越线 | slow_field / barrier / cryo | 纯经济卡 |
| 已有 split_shot + multishot | pierce / critical / recycle | 毒/冰补覆盖 |

### 2.6 一局内技能成长示意

```
出门：先锋的 sig_railvolley(Lv2) + sig_overload(Lv1) + 自带 skill_split_shot(Lv3)
              [专属]                [专属]              [通用格1]
       通用格2 = 空   通用格3 = 空

局内：
  杀怪攒满第1管 → 三选一(空槽优先)：出现 pierce / cryo / gold_rush，选 skill_pierce 进通用格2
  杀精英 → 额外三选一：出现 incendiary / recycle / split_shot升级，锁定 incendiary 后刷新，选 skill_recycle
  杀怪攒满第2管 → 三选一(升级)：split_shot run_level→4（触发分支：更多分裂）
  ...持续到通关，run_level 全部在关束清零
```

## 3. 每个技能的"独立成长路径"（分支树）

这是你要求的"每个技能都有自己的成长路径"。落地方式：**每个技能有一棵小分支树**，分两层控制：

- **局外（天赋点）**：花点数把 `base_level` 往上推，**解锁**该分支树上的节点（点亮可选项）。
- **局内（选卡）**：在已点亮的节点里，靠"升级卡"实际**走**某一条分支。

### 分支树通用结构（每个技能一致的骨架）

```
Lv1  基础效果
Lv2  数值强化（量变）
Lv3  ┬ 分支A（质变方向甲）
     └ 分支B（质变方向乙）   ← base_level≥3 才在局内出现 A/B 二选一
Lv4  延续所选分支再强化
Lv5  ┬ 终极进化X（接 A 线）
     └ 终极进化Y（接 B 线）   ← base_level=5 才能在局内进化到顶
```

> 具体每个技能的 Lv1~Lv5 与 A/B/X/Y 内容，逐条写在 `05_skills.md`。
> 数据结构（节点、解锁条件、效果字段）见 `data/schema.md` 的 `skills.json`。

## 4. 与货币体系闭环（防"升不动" / 防"碾压"）

你的两个硬要求在这里兑现：

### 4.1 保证升得动

- 天赋点来自每次升级（前期 1/级，后期 2/级），**全程产出 ≈ 全技能升满所需的一定富余**——但不是一次给够，玩家要**选择**优先点哪些技能（流派取舍），这正是养成乐趣。
- 金币产出对齐"重刷 2~4 遍即可把核心装备升到建议档"（见 `09` 曲线）。
- 星产出（297 上限）> 全部玩法内容解锁成本（≈220），皮肤为可选盈余消费。
- **没有任何升级是被随机/付费门挡住的**：所有提升都靠可稳定获取的资源。

### 4.2 保证不碾压

- 关卡难度按"**建议等级 + 合理装备 + 合理 build**"标定（见 `08`/`09`）。
- 即使顶配，Boss 仍有机制需要应对（免疫属性要求提前准备替代流派、特殊行为需要操作），数值再高也得"打对"。
- 养成提供的是"**让你够得着**"，不是"**让你躺赢**"：满配玩家拿 3 星仍需基本操作，欠养成玩家靠极限操作也能拿 1~2 星。

## 5. 角色养成差异

4 个角色各自独立的等级、天赋点、技能 base_level、专属技分支（互不共享）。
- 切换角色 = 切换一整套养成（鼓励玩家深耕 1~2 个，也允许全练）。
- 金币、星、装备是**账号共享**的（跨角色通用），只有"角色等级/技能/专属技"按角色独立。
- 角色定位、专属技与基础成长曲线见 `04_characters.md`。

## 6. 玩家长期目标（动机设计）

| 时间尺度 | 目标 |
|---|---|
| 每一局 | 拿更高星、掉更多金币、攒经验升级 |
| 每天/每段 | 把某件装备强化到下一档、点亮某个心仪技能分支 |
| 中期 | 解锁第 2/3/4 个角色，尝试不同元素流派 |
| 长期 | 99 关全三星、解锁全部内容、挑战变异尸潮/无限尸潮、收集外观与章节徽章、打磨极限 build |
