# Release Notes Draft

## 1.0.0

- 99-level offline campaign.
- Fixed-bottom defense shooting with manual aim, target lock, and target-priority switching.
- Four playable characters, eight weapons, armor, chips, pets, and skill collection pages.
- 16-card in-run skill pool with projectile, element, control, shield, economy, and burst upgrades.
- Enemy roster includes runners, tanks, support units, shield units, summoners, phase enemies, regenerators, splitters, and Boss fights.
- Progression with level unlocks, stars, gold, equipment selection, pets, and weapon upgrades.
- Local save only. No account required.
