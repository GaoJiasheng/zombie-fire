Zombie Fire Support

Zombie Fire is an offline vertical shooter / tower-defense roguelite.

Before Contacting Support

Please include the following information when reporting a problem:

  • Device model and operating system version.
  • Game version and build number.
  • Level number and selected character, weapon, armor, chip, and pet.
  • Steps to reproduce the issue.

Data Collection

The current build is offline and does not collect personal data, use ads, use analytics, or include in-app purchases.

Support Channel

Email gaojiasheng.him@foxmail.com for private support requests.

You can also open a request in the Zombie Fire public issue tracker. Do not include personal or sensitive information in a public report.
