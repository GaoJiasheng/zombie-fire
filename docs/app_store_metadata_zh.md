## Description

Zombie Fire 是一款竖屏尸潮防线射击游戏。你将操控底部自动机炮，对抗从城市废墟中涌来的僵尸。

在每一局中，你需要调整炮口方向、锁定高威胁目标，并在升级时从三张技能卡中选择强化。分裂、穿透、多重、元素、屏障和收益类技能会改变你的火力节奏。

当前版本包含 99 个关卡、4 名角色、8 种武器、护甲/芯片/宠物养成，以及多种僵尸和 Boss 机制。

特点：

- 竖屏单手友好的防线射击。
- 自动开火 + 手动瞄准。
- 双击锁定危险目标。
- 局内三选一技能构筑。
- 角色、武器、护甲、芯片、宠物组成局外配置。
- 99 关尸潮防线挑战。
