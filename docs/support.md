# Zombie Fire Support Draft

This repository includes a static support page draft at `docs/public/support.html`. Host it on a public HTTPS site before App Store submission and paste that URL into App Store Connect.

Required before submission:

- Public support URL.
- Public privacy policy URL.
- App Store Connect app name, subtitle, category, age rating, screenshots, and review notes.
- Contact channel for player issues.

Current local support note:

- App: Zombie Fire
- Genre: vertical shooter / tower-defense roguelite
- Offline play: yes
- Data collection: none in current build
