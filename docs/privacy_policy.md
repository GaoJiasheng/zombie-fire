# Zombie Fire Privacy Policy Draft

Last updated: 2026-06-25

Zombie Fire does not currently collect, store, sell, or share personal data.

The current build stores gameplay progress locally on the player's device, including level stars, gold, experience, and weapon upgrade level. This save data is not transmitted to a server.

The current build does not include analytics, advertising SDKs, account login, purchases, push notifications, or third-party tracking.

If future versions add online services, analytics, ads, purchases, leaderboards, cloud saves, or account features, this policy must be updated before release.

Support: publish the support page in `docs/public/support.html` and use its public URL in App Store Connect.
